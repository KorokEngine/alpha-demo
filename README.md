# alpha-demo
demos for korok engine's alpha release (will be deleted later)

To install korok engine: 
```
go get -insecure korok.io/korok
```
Clone demo project:
```
git clone git@github.com:KorokEngine/alpha-demo.git
```
Run demo(eg:text):

```
go run text/main.go
```
